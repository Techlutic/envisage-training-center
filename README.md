## 🌟 Overview

A luxury, competition-crushing landing page for Envisage Training Academy - UK's premier aesthetics training institution. Built with cutting-edge 2024 web design trends, premium animations, and unique interactive elements.

## ✨ Features

- 🎨 **Premium Design** - Luxury aesthetic with gold accents & glass morphism
- ⚡ **Performance Optimized** - Lazy loading, code splitting, GPU-accelerated animations
- 📱 **Fully Responsive** - Mobile-first design for all devices
- ♿ **Accessible** - WCAG 2.1 AA compliant with reduced motion support
- 🎭 **Rich Animations** - Scroll reveals, parallax, magnetic buttons, custom cursor
- 🔍 **SEO Ready** - Structured data, meta tags, semantic HTML
